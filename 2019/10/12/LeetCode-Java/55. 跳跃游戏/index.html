题目描述 **

给定一个非负整数数组，你最初位于数组的第一个位置。

数组中的每个元素代表你在该位置可以跳跃的最大长度。

判断你是否能够到达最后一个位置。

示例 1:

输入: [2,3,1,1,4]
输出: true
解释: 从位置 0 到 1 跳 1 步, 然后跳 3 步到达最后一个位置。

示例 2:

输入: [3,2,1,0,4]
输出: false
解释: 无论怎样，你总会到达索引为3的位置。但该位置的最大跳跃长度是0，所以你永远不可能到达最后一个位置。

题解

解法一（自己想的贪心）

从后往前，不为0的位置一定能走到最后，只需考虑为0的位置

class Solution {
    public boolean canJump(int[] nums) {
        if(nums.length == 1) return true;
        int zeroPosition = -1;
        for(int i=nums.length-1; i>=0; i--) {
            if(zeroPosition == -1) {
                if(nums[i] != 0) continue;  // 不考虑
                else {
                    zeroPosition = i;  // 记下当前0位置
                    continue;
                }
            }
            if(zeroPosition == nums.length-1) {
                if(zeroPosition - i <= nums[i]) {  // 如果0是最后位置，则需大于等于该位置
                    zeroPosition = -1;
                }
            }
            else {
                if(zeroPosition - i < nums[i]) {  // 否则需要大于该位置
                    zeroPosition = -1;
                }
            }

        }
        return zeroPosition == -1;
    }
}

解法二 官解贪心

从右向左迭代，对于每个节点我们检查是否存在一步跳跃可以到达 GOOD 的位置（currPosition + nums[currPosition] >= leftmostGoodIndex）。如果可以到达，当前位置也标记为 GOOD ，同时，这个位置将成为新的最左边的 GOOD 位置，一直重复到数组的开头，如果第一个坐标标记为 GOOD 意味着可以从第一个位置跳到最后的位置。

class Solution {
    public boolean canJump(int[] nums) {
        int lastPos = nums.length - 1;
        for(int i=nums.length-1; i>=0; i--) {
            if(nums[i]+i >= lastPos) lastPos = i;
        }
        return lastPos==0;
    }
}
